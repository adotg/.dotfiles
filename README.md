# dotfiles

From home `~`

Installs bash vim tmux
```bash
.dotfiles/install.sh
```

To only install tmux
```bash
.dotfiles/install.sh tmux
```

Support local config
- for bash `~/.bash_profile.local`
- for vim `~/.vimrc_local`
- for tmux `~/.tmux/tmux.local.config`
